Mastermind

This is a simple implementation of the Mastermind game. Mastermind is a guessing game designed to test your deduction skills. Play by clicking the colored dots.

  • 1Select first color from color palette
  • 2Select second color from color palette
  • 3Select third color from color palette
  • 4Select fourth color from color palette
  • 5Select fifth color from color palette
  • 6Select sixth color from color palette
  • ⌫Remove the last selected color
  • ↵Submit your guess
  • EscClose the dialog
